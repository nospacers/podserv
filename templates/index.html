How downloads work

Download to Pi stores the episode on the Raspberry Pi server so it is always available on your home network. Cache on phone asks the app to store that downloaded file in browser storage for offline playback.

Pi deployment

Run this on the Pi, keep it on the same Wi‑Fi network as your phone, and install the PWA from Chrome. For a more durable setup, run it behind a systemd service on boot.
